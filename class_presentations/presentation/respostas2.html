Exercise 7 – Answer

We have the MTH:

$$H_0^{T}(t) = R_z(\omega_At)D_x(d_{TS})R_z((\omega_D-\omega_A)t).$$

It may seem curious that the left multiplication is \(R_z((\omega_D-\omega_A)t)\), not \(R_z(\omega_Dt)\). The thing is, if we did it that way, the resulting rotation matrix of the MTH would be a rotation in \(z\) by \((\omega_A+\omega_D)t\). That implies that the angular velocity for Earth rotating around itself would be \(\omega_A+\omega_D\), which is not what was specified.

Exercise 8 – Answer

Yes. For this, we need \(Q^TQ = QQ^T = I_{3 \times 3}\) and \(\det(Q)=+1\).

Note that \(Q^T = I_{3 \times 3}-\epsilon A\) (we used the antisymmetry property of \(A\) and the linearity of transposition). Then \(Q^TQ = I_{3 \times 3} - \epsilon A + \epsilon A - \epsilon^2 A^2\). Since we consider \(\epsilon^2 = 0\), we have \(Q^TQ = I_{3 \times 3}\). Similarly, we can prove \(QQ^T = I_{3 \times 3}\).

For \(\det(Q)=+1\), you can write down the matrix \(I_{3 \times 3} + \epsilon A\) with \(A\) antisymmetric, compute its determinant, and put \(\epsilon^2=0\) to see it becomes +1.

Note: This (seemingly) simple result is linked to a much deeper theory called Lie Theory, illustrating an important geometric property of the space of rotations.

Exercise 10 – Answer

Yes. Every rotation is a rotation about some axis. Let \(r\) be a nonzero vector lying on the rotation axis of matrix \(Q\). Then \(r' = Qr\) represents the point \(r\) after the rotation about that axis. Because that point is already on the axis, nothing changes, and so \(r' = r = Qr\).

Example: rotating the vector \(r = (0\ 0\ 1)^T\) about the \(z\)-axis leaves it unchanged.

Exercise 12 – Answer

If \(Q_A^B\) and \(s_A^B\) are the rotation matrix and translation vector extracted from \(T_A^B\), then \(p_A = Q_A^B\,p_B + s_A^B\). In other words, to change coordinates of a vector from frame \(\mathcal{F}_B\) to \(\mathcal{F}_A\), we apply the rigid transformation from \(\mathcal{F}_A\) to \(\mathcal{F}_B\) to the point written in \(p_B\).

Exercise 13 – Answer

This problem is a particular case of the reference frame transformation problem explained here.

The logic is as follows: consider a third frame \(\mathcal{F}_C\) whose center is this point \(P\) (its orientation is irrelevant here). Let \(T_B^C\) be the transformation matrix between \(\mathcal{F}_B\) and \(\mathcal{F}_C\). We know, from the two interpretations of MTHs, that \(T_B^C\) is frame \(\mathcal{F}_C\) written in \(\mathcal{F}_B\). Hence the translation part of \(T_B^C\) is precisely \(p_B\).

By exactly the same logic, let \(T_A^C\) be the transformation between \(\mathcal{F}_A\) and \(\mathcal{F}_C\). Its translation part is precisely \(p_A\).

But we know \(T_A^C = T_A^B\,T_B^C\). If we read only the translation part of that equation, we reach the desired result.
